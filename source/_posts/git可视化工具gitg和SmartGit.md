## 安装gitg工具：

安装命令：

```bash
sudo apt install gitg
```

## 使用方法：

输入gitg命令会自动跳转下图中，然后按下Crtl+C才能够退出来。我是因为意外之前添加一个分支线，所以pull --rebanse总是会有冲突，故用此查看，回退（即撤销到）到分支岔口，然后在pull同步，以后在pull就不会有冲突出现了
